# FastAPI + Vercel

This projects used fastapi to build a very simple qr code generator api.
The api will be included in Fehmarn-Yoga.de to help the creation of lasting and free qr-codes.


## Running Locally

```bash
npm i -g vercel
vercel dev
```

Your FastAPI application is now available at `http://localhost:3000`.
